Important files are listed below: 

run_palm.R: the script that can run PALM on this dataset. And you can change the starting dimension here. 

log_likelihood_on_test_data.R: this is the script for getting the log-likelihood for unseen data (test dataset). 

Time-complexity-ams.R: getting the runtime. 

ipd.R: Running IPD on AMS dataset, which is used in the response letter. 

Note: for visualizations, check the folder "collect-res".
